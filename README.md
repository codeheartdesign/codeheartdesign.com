## How to run

Install dependencies with preferably [`yarn`](https://yarnpkg.com/) and run one of the following commands:

| Command      | Description                                       |
|--------------|---------------------------------------------------|
| `yarn build` | Build css and the static site                     |
| `yarn watch` | Build, watch for changes and start a local server |
